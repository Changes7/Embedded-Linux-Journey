# docs — 学习笔记

本目录用于存放基于 **STM32MP157** 的嵌入式 Linux 学习笔记，采用 Markdown 格式编写。

## 目录用途

- 记录开发板环境搭建、交叉编译、文件系统与启动流程等学习过程
- 整理应用层编程、驱动开发、设备树与内核调试等知识点
- 归档实验步骤、问题排查记录与参考资料链接

## 开发规范

1. **命名**：使用小写字母与连字符，如 `01-cross-compile-setup.md`、`gpio-driver-notes.md`
2. **结构**：每篇笔记应包含标题、日期、目标与结论；复杂主题可拆分为系列文档
3. **代码示例**：注明运行环境（主机 / 开发板）与依赖版本，代码块需标注语言类型
4. **图片与附件**：统一放在 `docs/assets/` 子目录，笔记中通过相对路径引用
5. **版本同步**：涉及内核、U-Boot 或 Yocto 版本变更时，在文首注明对应版本号
