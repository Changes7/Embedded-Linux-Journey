# scripts — 自动化脚本

本目录存放 **STM32MP157** 嵌入式 Linux 开发流程中的自动化脚本。

## 目录用途

- 开发环境初始化与交叉编译工具链安装
- 镜像构建、内核编译、设备树生成等一键化流程
- NFS 部署、串口日志采集、开发板远程调试等运维脚本
- CI/CD 或本地重复性任务的封装

## 开发规范

1. **脚本类型**：Shell（`.sh`）为主；Python 脚本需注明最低版本及依赖包
2. **可移植性**：使用 `#!/usr/bin/env bash` 作为 Shebang；路径优先通过变量或参数传入，避免硬编码绝对路径
3. **错误处理**：开启 `set -euo pipefail`（Bash）；关键步骤失败时应输出明确错误信息并以非零状态码退出
4. **文档说明**：每个脚本文件头部注释用途、用法示例及所需环境变量
5. **权限管理**：脚本本身不提交可执行位依赖；在 README 或注释中说明 `chmod +x` 需求
6. **安全性**：禁止在脚本中明文存储密码或密钥；敏感配置通过环境变量或外部配置文件读取，并确保该配置文件已加入 `.gitignore`
